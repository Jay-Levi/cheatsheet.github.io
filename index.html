< table > The < table > element in HTML is used to present data in a table made of rows and columns MDN
< thead > The < thead > element defines a set of rows defining the head of the columns of the table. MDN
< tbody > The < tbody >element encapsulates a set of table rows (< tr > elements), indicating that they comprise the body of the table MDN
< tr > The < tr > element defines a row of cells in a table. after a row has been created, content can be added to individual cells using the < td > and < th > elements MDN
< td > The < td > element defines a cell within a table. Its column can be expanded using the < col > element with the span attribute. MDN
